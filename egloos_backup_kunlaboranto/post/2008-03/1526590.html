[sqlldr] row 구분자 지정 (CR)

altibase 와 oracle에서 row 구분자에 CR를 주는 방법

# iloader (altibase)
  \n을 %n, \t를 %t 로 표시한다. 최대 10자의 조합이 가능

  -r "%nR-r%n"
  -t "^C-c^"

# sqlldr

  infile 'example.dat'  "str '\nR-r\n'"
  fields terminated by '^C-c^'
