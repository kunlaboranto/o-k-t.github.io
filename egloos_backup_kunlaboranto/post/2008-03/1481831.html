Gloups (IS/IT--Management)      11 Jul 05 11:07

Dont forget to pass those commands before using nmon

chmod ugo+r /dev/kmem
and
chdev -l sys0 -a iostat=true

I tried on an AIX 5.2 Box and all is ok
