From : http://database.hanbat.ac.kr/attach/DatabaseProgramming/Chapter09.pdf
          자바 + DB 기초 강의

1)
int executeUpdate(String sql): 명령문이 영향을 끼친 row 개수 (변경개수) 를 리턴

2)
boolean execute(String sql): 첫번째 결과가 ResultSet이면 true, 결과가 없거나 변경개수이면 false를 리턴
  • ResultSet getResultSet(), int getUpdateCount(), boolean getMoreResults() 

  모든 종류의 SQL 명령문
  - 미리 알려지지 않은 동적 SQL문
  - 여러 개의 값을 반환하는 스토어드 프로시저

3)
따라서, wasNull()메소드를 호출하여
• getFloat()의 데이터가 0인지 NULL인지 확인해야 함

4)
필드 최대 크기
– Statement 객체의 setMaxFieldSize(), getMaxFieldSize()를 사용
– 0을 설정하면 필드 최대 크기는 제한을 두지 않게 됨
– 다음의 SQL 타입에 해당하는 칼럼에 대해서만 적용가능
• BINARY VARBINARY LONGVARBINARY
• CHAR VARCHAR LONGVARCHAR

5)
쿼리 시간 제한
– Statement 객체의 setQueryTimeout(), getQueryTimeout()를 사용
– 시간제한 값의 단위는 초(second)임

6) 스크롤 가능한 ResultSet 객체 생성하기

전진과 읽기만 가능한 ResultSet 객체를 생성하기 위한 Statement 객체 생성
– Statement stmt = conn.createStatement();
– Statement stmt = conn.createStatement(ResultSet.TYPE_FORWARD_ONLY,
ResultSet.CONCUR_READ_ONLY);
