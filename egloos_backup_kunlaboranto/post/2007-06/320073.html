vsftpd 설정 - This FTP server is anonymous only

[root@localhost vsftpd]# ftp localhost
Connected to localhost.localdomain.
220 (vsFTPd 1.2.0)
530 Please login with USER and PASS.
530 Please login with USER and PASS.
KERBEROS_V4 rejected as an authentication type
Name (localhost:root): root
530 This FTP server is anonymous only.
Login failed.

 

환경파일의 위치 찾기 (RHEL3.0에서 잘못된 경로, /etc/vsftpd/vsftpd.conf 에 위치함)

$ strings /usr/sbin/vsftpd |grep conf
/etc/vsftpd.conf

 

참조링크 :

http://coffeenix.net/board_view.php?cata_code=61&bd_code=89&bpage=

 

$ cat /etc/xinetd.d/vsftpd
service ftp
{
        disable         = no
        flags           = REUSE
        socket_type     = stream       
        wait            = no
        user            = root
        server          = /usr/sbin/vsftpd
        log_on_failure  += USERID
}
